
ATTENZIONE: questa e' una test suite scritta in JUnit 4, per eseguire i test e' necessario aggiungere le libreria JUnit4 e rimuovere JUnit 3

- Aggiungere JUnit 4
	- clic destro sul nome del progetto
	- Build Path
	- Add Libraries...
	- JUnit
	- Next
	- JUnit 4
	- Ok

- Rimuovere JUnit 3
	- clic destro sulla libreria JUnit 3 library
	- Build Path
	- Remove from Build Path





WARNING: this is a JUnit 4 test suite, to run the tests you need to add the JUnit4 library and remove JUnit 3 library:

- Add JUnit 4
	- right click on the project name
	- Build Path
	- Add Libraries...
	- JUnit
	- Next
	- JUnit 4
	- Ok

- Remove JUnit 3
	- right-click on JUnit 3 library
	- Build Path
	- Remove from Build Path
